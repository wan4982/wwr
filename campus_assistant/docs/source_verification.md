# 数据来源核验说明

`data/campus_data.csv` 已从最初的实训模拟数据改为“官网可核验口径”数据。

## 已参考的学校官网来源

- 安徽交通职业技术学院官网：https://www.acvtc.edu.cn/
- 教务处公开页面：《安徽交通职业技术学院学分制学籍管理办法》：https://www.acvtc.edu.cn/jwc/info/1031/1821.htm
- 教务处网站办事指南栏目：用于确认学生证补办、休学、复学等办事流程栏目存在。
- 学校官网公共服务栏目：用于确认校历入口存在。
- 财务处网站：用于确认网上缴费系统、收费公示栏目存在。

## 调整说明

- 删除或改写了未能从官网公开页面确认的模拟细节，例如“奖学金固定绩点线为3.0”“宿舍报修1至3个工作日处理”“一卡通挂失补办具体流程”等。
- 奖学金相关回答改为：官网能确认“平均学分绩点是评定奖学金的重要依据之一”，但具体最低绩点线、名额、材料要求应以学生处或学院当年通知为准。
- 宿舍报修和一卡通流程如果需要继续加入，应再找到后勤、校园卡中心或学生手册的公开权威来源后补充。
